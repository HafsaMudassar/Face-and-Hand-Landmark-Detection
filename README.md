# Craniofacial Landmark Capture

This project uses Mediapipe and OpenCV to detect facial and hand landmarks from a webcam. 
You can capture frames and save landmarks for craniofacial screening or further analysis.

## Features
- Live webcam feed with face and hand landmarks
- Capture frames by pressing `C`
- Save landmarks in JSON format by pressing `S`
- Exit program by pressing `Q`

## Requirements
- Python 3.11+
- OpenCV
- Mediapipe
